Plugin Documentation

Goals available for this plugin:

Goal Description
lombok:delombok Delombok java source with lombok annotations.
lombok:help Display help information on lombok-maven-plugin.
Call mvn lombok:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
lombok:testDelombok Delombok java test source with lombok annotations.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.0.4
JDK 1.6
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.projectlombok</groupId>
          <artifactId>lombok-maven-plugin</artifactId>
          <version>1.14.2.0</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.projectlombok</groupId>
        <artifactId>lombok-maven-plugin</artifactId>
        <version>1.14.2.0</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
