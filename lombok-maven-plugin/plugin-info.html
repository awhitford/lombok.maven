Plugin Documentation

Goals available for this plugin:

GoalDescription
lombok:delombokDelombok java source with lombok annotations.
lombok:helpDisplay help information on lombok-maven-plugin.
Call 
  mvn lombok:help -Ddetail=true -Dgoal=<goal-name>
to display parameter details.
lombok:testDelombokDelombok java test source with lombok annotations.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven2.2.1
JDK1.6
MemoryNo minimum requirement.
Disk SpaceNo minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.projectlombok</groupId>
          <artifactId>lombok-maven-plugin</artifactId>
          <version>0.10.6.0</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.projectlombok</groupId>
        <artifactId>lombok-maven-plugin</artifactId>
        <version>0.10.6.0</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
